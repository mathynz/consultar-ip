Política de Privacidade

Este site respeita a privacidade dos usuários. As informações coletadas são utilizadas apenas para aprimorar o serviço e melhorar a experiência de navegação.

Para análise e otimização do site, podem ser utilizados serviços de terceiros, como Google Analytics e Google Search Console, que ajudam a entender o uso do site e identificar oportunidades de melhoria.

As consultas de IP realizadas no site utilizam a API do ip-api.com. Nenhum dado pessoal é compartilhado com terceiros para fins comerciais, e o usuário permanece responsável pelo uso adequado das informações obtidas através deste site.
